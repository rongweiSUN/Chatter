# 当 随口说 X DeskClaw

> 用嘴巴驱动你的桌面 AI 助手。

---

## 它们是什么？

**随口说** 是一款 macOS 菜单栏语音输入工具。按下快捷键说话，松开即识别，文字自动粘贴到光标位置——取代打字，用说的。

**DeskClaw** 是一款本地运行的桌面 AI 助手。它能理解你的意图，调用工具，帮你完成各种任务。

当两者结合，**随口说成为 DeskClaw 的语音入口**——你不再需要打开 DeskClaw 窗口、敲键盘提问。长按快捷键，对着麦克风说一句话，DeskClaw 就开始工作。

---

## 能做什么？

### 语音输入（随口说单独使用）

- **短按快捷键**（默认右 Command），开始录音；再按一次，语音转文字并粘贴
- 选中一段文字后录音说"翻译成英文"，AI 自动改写并替换
- 内置技能引擎：口语过滤、结构优化、用户词典纠正、自定义 Prompt 技能

### 语音助手（随口说 X DeskClaw）

- **长按快捷键**，按住说话，松开后自动发送给 DeskClaw
- DeskClaw 处理完毕，结果以浮窗弹出，支持 Markdown 渲染、图片展示
- 浮窗底部可**继续打字对话**，或点击"在 DeskClaw 中继续对话"跳转到完整界面
- 多个回答窗口可同时存在，互不干扰

### 典型场景

| 你说的话 | 发生了什么 |
|---------|----------|
| "帮我总结一下剪贴板里的文章" | DeskClaw 读取剪贴板内容并生成摘要，浮窗展示 |
| "把这段话翻译成日语"（选中文字后） | AI 直接替换选中文字为日语翻译 |
| "查一下今天的天气" | DeskClaw 调用工具查询，浮窗展示结果 |
| "写一封请假邮件" | DeskClaw 生成邮件内容，你可以在浮窗中继续修改 |

## 怎么用？

### 1. 安装

确保 macOS 12 以上，Python 3.9 以上。

```bash
git clone <repo-url>
cd 语音助手
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

或者直接使用打包好的 `随口说.app`。

### 2. 启动

```bash
python main.py
```

菜单栏出现「随口说」，表示已就绪。

如果需要使用 DeskClaw 联动功能，确保 DeskClaw 应用已启动并在本地运行。

### 3. 授权

首次使用需在「系统设置 → 隐私与安全性」中授予：

- **麦克风** — 录音
- **辅助功能** — 模拟粘贴（Cmd+V）
- **输入监控** — 监听全局快捷键

### 4. 日常使用

**语音输入（短按）**

1. 光标放在任意输入框
2. 短按右 Command → 开始录音
3. 说话
4. 再按一次 → 识别、处理、粘贴

**语音助手（长按）**

1. 长按右 Command → 按住说话
2. 松开 → 发送给 DeskClaw
3. 浮窗弹出回答，可继续对话

**选中文字 + 语音指令**

1. 选中一段文字
2. 短按右 Command → 录音
3. 说出指令（如"改成更正式的语气"）
4. 再按一次 → AI 按指令改写并替换

**取消录音**

- 录音过程中按 ESC，或点击浮窗上的取消按钮

### 5. 配置

点击菜单栏「随口说」→ 打开主窗口：

- **模型** — 配置语音识别（ASR）和大语言模型（LLM）的服务商与密钥
- **技能** — 管理口语过滤、结构优化、用户词典、自定义技能
- **通用** — 修改快捷键、开关自动粘贴、开关悬浮窗

## 常见问题

**Q: 不安装 DeskClaw 能用吗？**
可以。随口说的语音输入、选中文字改写、技能处理等功能完全独立。DeskClaw 仅在长按助手模式下使用。

**Q: 快捷键可以改吗？**
可以。打开主窗口 → 通用设置，点击快捷键区域后按下你想要的按键即可。支持修饰键和普通按键。

**Q: 支持哪些语音识别服务？**
火山引擎、阿里云、SenseVoice（本地）、自定义 API。

**Q: 支持哪些大语言模型？**
OpenAI、Claude、DeepSeek、Gemini、通义千问、火山引擎（豆包）、Ollama（本地）、自定义 API。
